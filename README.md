# Java Studies Repository

Este repositório reúne atividades práticas e estudos iniciais com a linguagem Java.

## Organização

A ideia principal é manter cada estudo/desafio em uma branch separada. Assim, cada ramo pode conter:

- um conjunto de arquivos e exemplos específicos de um desafio;
- commits e histórico próprios para cada estudo;
- experimentos isolados sem misturar conteúdo no `main`/`master`.

> Observação: atualmente não há nada na `master` porque a proposta é usar branches independentes para cada desafio.


## Acesso aos Desafios
Para ver informações e detalhes de cada desafio, você deve mudar para a branch correspondente ao desafio e consultar os arquivos dessa branch.
