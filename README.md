# Java Studies

Este repositório contém exercícios e exemplos em Java organizados na pasta `geral`.

## Conteúdo

- `Calculadora.java` - exemplo de operações básicas de calculadora.
- `Calendario.java` - exemplo de manipulação de datas ou exibição de calendário.
- `Colecoes.java` - exemplos com coleções Java (`List`, `Set`, `Map`, etc.).
- `Endereco.java` - classe que representa um endereço.
- `Imc.java` - cálculo de IMC (Índice de Massa Corporal).
- `Menu.java` - exemplo de menu interativo de console.
- `Primeira.java` - projeto de exemplo inicial para aprendizado.

## Como compilar

No terminal, navegue até a pasta raiz do projeto e execute:

```powershell
cd d:\Estudos\programacao\java_studies\geral
javac *.java
```

## Como executar

Após compilar, execute a classe desejada. Por exemplo:

```powershell
java Primeira
```

ou

```powershell
java Menu
```

> Observação: este projeto usa o compilador Java padrão (`javac`) e deve funcionar com Java 8 ou superior.

## Estrutura

- `geral/` - contém os arquivos-fonte Java do estudo.

## Objetivo

Manter um conjunto de exemplos práticos para estudar lógica, sintaxe e recursos básicos de Java.
