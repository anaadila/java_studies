# Java Studies Repository

Este repositório reúne atividades práticas e estudos iniciais com a linguagem Java.

## Organização

A ideia principal é manter cada estudo/desafio em uma branch separada. Assim, cada ramo pode conter:

- um conjunto de arquivos e exemplos específicos de um desafio;
- commits e histórico próprios para cada estudo;
- experimentos isolados sem misturar conteúdo no `main`/`master`.

> Observação: atualmente não há nada na `master` porque a proposta é usar branches independentes para cada desafio.


## Acesso aos Desafios
Para ver informações e detalhes de cada desafio, você deve mudar para a branch correspondente ao desafio e consultar os arquivos e README dessa branch.


## Informações sobre os Desafios

### 1. 🏦 Desafio 'Banco':

- **Descrição**: Projeto simples de simulação de banco em Java para fins de estudo.
- **Visão geral**: Sistema bancário de linha de comando com funcionalidades de cadastro, listagem, operações financeiras e visualização de saldo total por categoria.
- **Funcionalidades**:
  - Cadastro de clientes e contas.
  - Listagem de clientes e contas.
  - Realização de operações financeiras (depósito, saque, transferência).
  - Visualização do saldo total por categoria (poupança, corrente, investimento).

### 2. 🚗 Desafio 'Carro':

- **Descrição**: Projeto simples de modelagem de um carro em Java para fins de estudo.
- **Visão geral**: Sistema de linha de comando para cadastro e exibição de informações de um carro, incluindo atributos como portas, chassi, ano, combustível e pneus.
- **Funcionalidades**:
  - Cadastro de atributos do carro via console.
  - Cálculo automático da quantidade total de pneus (incluindo step) e parafusos.
  - Exibição dos dados do carro, incluindo a cor (opcional).

### 3. 🧑‍🎓 Desafio 'Chamada':

- **Descrição**: Projeto simples de sistema de chamada em Java para fins de estudo.
- **Visão geral**: Sistema de linha de comando para registrar presença de alunos em chamadas, com funcionalidades de cadastro de alunos, listagem e verificação de presença por data.
- **Funcionalidades**:
  - Cadastro de alunos via console.
  - Listagem de alunos cadastrados.
  - Registro de presença em chamadas por data.
  - Verificação do status de presença por data.

### 4. 🛍️ Desafio 'Contas':

- **Descrição**: Projeto simples de gerenciamento de clientes e compras em Java para fins de estudo.
- **Visão geral**: Sistema de linha de comando para cadastro de clientes, registro de compras e análise de dados de clientes, incluindo autenticação por nome e senha.
- **Funcionalidades**:
  - Cadastro de clientes via console.
  - Adição automática de 10 clientes.
  - Listagem de todos os clientes cadastrados.
  - Identificação do cliente com mais compras.
  - Identificação do cliente com menos compras.
  - Cálculo da média de compras dos clientes.
  - Autenticação de clientes por nome e senha.

### 5. 📦 Desafio 'Produtos':

- **Descrição**: Projeto simples para gerenciar produtos para fins de estudo.
- **Visão geral**: Sistema de linha de comando para cadastro e gerenciamento de produtos, estoque, usuários e pedidos de venda.
- **Funcionalidades**:
  - Cadastro de produtos.
  - Listagem de produtos cadastrados.
  - Cadastro de usuários ADMIN e ClIENTE.
  - Listagem de usuários cadastrados.
  - Cadastro de estoque para produtos.
  - Venda de produtos, com verificação de estoque disponível.
  - Listagem de pedidos de venda realizados.

### 6. ♾️ Desafio 'General':

- **Descrição**: Repositório geral com diversos exercícios e exemplos em Java para fins de estudo.
- **Visão geral**: Pasta geral contendo arquivos Java com exemplos de operações básicas, manipulação de datas, coleções, representação de endereço, cálculo de IMC, menu interativo e projeto inicial.
- **Funcionalidades**:
  - Calculadora: operações básicas de calculadora.
  - Calendário: manipulação de datas ou exibição de calendário.
  - Coleções: exemplos com coleções Java (List, Set, Map, etc.).
  - Endereço: classe que representa um endereço.
  - IMC: cálculo de IMC (Índice de Massa Corporal).
  - Menu: exemplo de menu interativo de console.
  - Primeira: projeto de exemplo inicial para aprendizado.
